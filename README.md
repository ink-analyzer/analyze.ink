# ink! Analyzer

This repository contains the source code for the [ink! Analyzer](https://analyze.ink/) website.

### Installation

```
$ yarn
```

### Local Development

```
$ yarn start
```

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

### Build

```
$ yarn build
```

This command generates static content into the `build` directory and can be served using any static contents hosting service.

### Deployment

This repo is configured to deploy to GitHub pages via a [GitHub actions workflow definition](/.github/workflows/pages.yml).
